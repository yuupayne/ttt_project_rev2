# CUI 〇×ゲーム 仕様書

## 1. 概要
- ターミナル上で動作するCUIの〇×ゲームを作成する。
- AIは用意せず、プレイヤー2名が交互に操作する。
- C++学習目的のため、コード内に日本語の解説コメントを付ける。

## 2. ルール
- 先手：〇（Player1）
- 後手：×（Player2）
- 勝利条件：縦・横・斜めに3つ揃ったら勝利。
- 引き分け条件：盤面が全て埋まった状態で勝者なし。
- 引き分け時表示：「引き分け」とだけ表示。

## 3. 表示仕様（CUI描画）
- 盤面は3×3固定。
- 空白マスはスペースで表示。
- 表示形式は5×5で、区切りは縦横線＋十字線を使用。
- 各ターンごとに盤面を再描画する（追記方式ではない）。

## 4. 入力仕様
- 入力は1〜9の数字指定（マスに番号を割り振る）。
- それ以外の入力は無効として再入力させる。
- 既に配置済みのマスを選んだ場合は警告表示し、再入力させる。
- 不正入力が連続しても特別な処理はしない（再入力のみ）。

## 5. ゲーム進行
- プレイヤー名は固定（Player1 / Player2）。
- ターン表示：「Player1の番です」「Player2の番です」。
- 勝利時表示：「勝者名（記号）」を表示。
- ゲーム終了後は次のゲームを開始する（再戦確認なし）。
- 先手は常にPlayer1固定。
- 再戦時には盤面を初期化。

## 6. 実装方針に関する補足
- 盤面の保持形式、勝利判定関数、表示／入力分離などは任意。
- C++学習用のため、クラスやソースファイルは過度に増やさず、シンプルな構成を優先する。
